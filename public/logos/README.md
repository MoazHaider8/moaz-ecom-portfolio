# Logo Files

Add your 7 company logos here with the following names:

- `logo1.png`
- `logo2.png`
- `logo3.png`
- `logo4.png`
- `logo5.png`
- `logo6.png`
- `logo7.png`

## Requirements:
- Format: PNG (preferred) or JPG/SVG
- Recommended size: 200-400px width, transparent background
- The logos will be displayed at 60px height and auto-scaled width
- Logos will be automatically converted to white color via CSS filter

## Example:
If you have company logos like:
- Client Company A → save as `logo1.png`
- Client Company B → save as `logo2.png`
- etc.

Just drop your 7 logo files into this folder and they'll automatically appear on the homepage!
